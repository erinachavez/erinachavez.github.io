This is my second go at creating a website devoted to muggle quidditch, this time utilizing bootstrap for the first time. This website was created while keeping in mind that the International Quidditch Association has a volunteer position open for a graphic designer.

BOOK COVER REDESIGN

MAGAZINE COVER AND LOGO PAIR

I was asked to create a minimal magazine cover and logo pairing as if I were a furniture designer. No doubt a simple and pointed experiment that yielded a similar result.

LACUNA INC. LOGO

This logo was created to represent the fictional company, Lacuna Inc., featured in the movie Eternal Sunshine of the Spotless Mind.

DYPTICH EVENT POSTERS

As the final project for my first graphic design course, I was asked to create a pair of dytich posters for a fictional event of my choosing. At the same time, we were asked to feature a certain typeface. In my case, I chose to focus on the Doves Type, creating a fictional event and posters to represent it.

STUPID DECISIONS MOVIE POSTERS

The purpose of this project was to create both a teaser and official movie poster for a fictional movie, based off of a book, song, poem, etc.. In this case, I chose to create the posters based off of the song Stupid Decisions by west coast punk band FIDLAR.
